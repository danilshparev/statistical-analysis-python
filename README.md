# Statistical Analysis with Python

Набор учебных проектов по математической статистике, выполненных на Python.

## Проекты

### Регрессионный анализ и прогнозирование
Построение и сравнение регрессионных моделей, оценка качества прогнозирования и статистическая проверка значимости модели.

### Проверка статистических гипотез
Проверка статистических гипотез с использованием параметрических и непараметрических критериев.

### Идентификация распределений и оценка параметров
Определение типа распределения выборки, оценка параметров и проверка согласия с теоретическими распределениями.
