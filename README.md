# Проекты по математической статистике

Набор учебных проектов по математической статистике, выполненных на Python.

## Проекты

### Определение теоретических законов распределения случайных величин на основе выборочных данных
Построение эмпирической функции распределения и гистограмм плотности вероятности для трех независимых признаков.
    * Точечная оценка параметров распределений двумя методами: **Методом Моментов (ММ)** и **Методом Максимального Правдоподобия**.
    * Сравнительный анализ теоретических моделей: Равномерное, Нормальное, Экспоненциальное.
    * Расчет статистических характеристик для сгруппированных (интервальных) данных.
    * Построение точных и асимптотических **доверительных интервалов** для параметров распределения.

### Проверка статистических гипотез
Сравнение двух независимых выборок: Проверка гипотезы о равенстве математических ожиданий. Использован двухвыборочный t-критерий Стьюдента и непараметрический критерий Манна-Уитни.
    * Проверка гипотезы о соответствии дисперсии выборки заданному эталонному значению ($\sigma^2_0 = 157.65$) с помощью критерия Хи-квадрат.
    * Проверка гипотезы согласия эмпирических данных с теоретическим показательным распределением с использованием критерия согласия Пирсона.

### Регрессионный анализ
Визуальный анализ взаимосвязи (построение Scatter Plot) площади участка и итоговой стоимости недвижимости.
    * Построение и сравнение трех типов регрессионных моделей: линейной, квадратичной и степенной.
    * Оценка качества моделей по метрикам: коэффициент детерминации, средняя квадратическая ошибка, средняя ошибка аппроксимации.
    * Глубокий анализ остатков линейной модели: расчет RSS, несмещенной оценки дисперсии ошибок, построение доверительных интервалов для коэффициентов регрессии.
    * Точечное и интервальное прогнозирование стоимости объекта для заданного значения площади.
